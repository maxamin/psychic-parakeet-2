HacmeCasino - Project Overview

Mission and Scope

What problem does this project address?
This project addresses the lack of real-world examples of application security for various frameworks. Currently, there are no Ruby on Rails or Ajax enabled application security training tools.
What is the goal of this project?
The goal of this project is to be a training tool that demonstrates several application security defects using Ruby on Rails with AJAX functionality. It is a part of the HacmeInc series of applications and should share a similar look and feel. Additionally, it should be a fully functional online gambling site! Also, this application should have a demo (HacmeBlackjack) up and running by May 10, 2006.
What is the scope of this project?
The scope of this project is to create a limited amount of security vulnerabilites, comparable to other HacmeInc applications, with a limited amount of functionality, also comparable to other HacmeInc applications.
What development methodology is being used?
See the software development methodology document.
Where should a new team member start?
For more information, see the project proposal.

Status

The status of this project is under consideration.

Once the documentation is developed, the code development will begin. The goal of reaching the May 10 deadline with a workable demo is under consideration.
